08 Clearspace Rule

The minimum clearspace around any logo variant is equal to the cap-height of the letterforms. No other element may enter this zone.

Dashed green border = minimum clearspace boundary. The zone scales proportionally with the logo size.

09 Incorrect Usage
✗ Do not stretch

Never distort aspect ratio. Use only as supplied.

✗ Do not rotate

The logo is always horizontal. No rotation, ever.

✗ Do not recolour

Never change brand colours. Only approved variants permitted.

✗ Do not add effects

No glows, shadows, gradients, or filters on the logo.

✗ Do not place on busy backgrounds

Only use on solid colour backgrounds from the approved palette.

✗ Do not use below minimum size

Wordmark minimum is 14px tall. Mark minimum is 12px tall.

10 Usage Rules
✓ Siempre
  • Usar únicamente los archivos SVG del diseñador — nunca recrear el logo en HTML o CSS
  • Usar la variante correcta según el fondo: oscuro para fondos claros, claro para fondos oscuros
  • Mantener la proporción original — nunca redimensionar solo en un eje
  • Respetar la zona de espacio libre equivalente a la altura de las mayúsculas
  • Usar el wordmark en navegación principal y el T mark como icono/avatar
  • El tamaño mínimo del wordmark es 14px de altura
✗ Nunca
  • Rotar, estirar, comprimir o deformar el logo de ninguna forma
  • Cambiar los colores — solo están permitidas las variantes aprobadas
  • Añadir sombras, brillos, gradientes u otros efectos al logo
  • Colocar el logo sobre fondos de textura, fotografías o patrones
  • Usar el logo por debajo del tamaño mínimo aprobado
  • Recrear el logo con tipografía — siempre usar el SVG oficial
